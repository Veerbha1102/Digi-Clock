# Digital Clock - Multiple Time Zones

A beautiful, responsive web application that displays the current time in different time zones around the world.

## Features

✨ **Multiple Time Zones**: Pre-configured clocks for 8 major cities worldwide
- New York (EST/EDT)
- London (GMT/BST)
- Tokyo (JST)
- Sydney (AEDT/AEST)
- Dubai (GST)
- Singapore (SGT)
- Los Angeles (PST/PDT)
- Mumbai (IST)

🌍 **Custom Time Zones**: Add any timezone supported by the browser
⏱️ **Real-time Updates**: Clocks update every second
🎨 **Beautiful UI**: Modern gradient design with smooth animations
📱 **Responsive Design**: Works on all devices (desktop, tablet, mobile)
⚡ **No Dependencies**: Pure HTML, CSS, and JavaScript

## How to Use

1. Open the application
2. Add custom time zones by clicking the "+Add Custom Time Zone" button
3. Enter any valid timezone (e.g., Asia/Bangkok, America/Toronto)
4. Remove custom zones by clicking the × button on the card

## Supported Timezones

The app supports all IANA timezone identifiers. Some examples:
- Asia/Bangkok
- Europe/Paris
- America/Toronto
- Africa/Cairo
- Australia/Melbourne
- Pacific/Auckland

For a complete list, check: https://en.wikipedia.org/wiki/List_of_tz_database_time_zones

## Browser Compatibility

- Chrome/Edge: ✅ Full support

## Technical Details

- **Language**: HTML5, CSS3, JavaScript (ES6+)
- **API**: Intl.DateTimeFormat (built-in browser API)
- **Styling**: CSS Grid, Flexbox, Gradients
- **Responsiveness**: Mobile-first design
